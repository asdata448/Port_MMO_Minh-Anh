# Võ Thị Minh Anh - HR Portfolio

Portfolio website của Võ Thị Minh Anh - HR Executive chuyên nghiệp.

## Tech Stack

- **Next.js 14** - React framework with App Router
- **TypeScript** - Type safety
- **Tailwind CSS** - Utility-first CSS framework
- **Framer Motion** - Animation library
- **React Intersection Observer** - Scroll-based animations

## Features

- ✨ Responsive design
- 🌸 Sakura/cherry blossom decorations
- 🎨 Pink-dominant color scheme from provided palette
- 🚀 Performance optimized
- 🔍 SEO optimized
- 📱 Mobile-friendly
- ♿ Accessible

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build

```bash
npm run build
```

### Start Production

```bash
npm start
```
